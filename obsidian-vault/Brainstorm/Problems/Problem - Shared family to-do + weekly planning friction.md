# Problem - Shared family to-do + weekly planning friction

## Who has this problem?
- Couples / families managing household tasks

## Pain
- Hard to find a shared system that supports weekly planning + daily execution
- Most apps get some features right but miss key ones

## Evidence
- Reddit thread: https://www.reddit.com/r/ProductivityApps/comments/1re5bw7/tried_15_todo_apps_still_havent_found_the_one/

## Requirements (from the post)
- Weekly / calendar view
- Shared lists (real-time sync)
- Recurring tasks
- Works on Android and PC
- Clean, minimal interface (not overwhelming)
- Free or reasonably priced
- Bonus: good home screen widget

## Top responses (top comments)
1) Suggestion: **use a shared calendar as the core system**
   - Mentions: recurring events/tasks, weekly visibility, add notes/files, share with partner, use colors/travel times.

2) Suggestion: **TickTick** (one-liner recommendation)

3) Alternative workaround: **dummy Gmail + Google Keep tasks + shared calendar**
   - Claim: can cover most of the requested features.
