# Idea - Family weekly planning todo + calendar hybrid

## Solves
- [[Problem - Shared family to-do + weekly planning friction]]

A shared “Sunday planning” workflow for couples/families:
- plan the week together
- stay aligned daily

## MVP scope
- Weekly view + calendar blocks
- Shared lists with real-time sync
- Recurring tasks
- Android + web (PC)
- Minimal UI

## Differentiator hypothesis
Make the weekly planning ritual the center (not just another task list).
